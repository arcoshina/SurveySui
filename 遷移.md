Windows 環境遷移指南
1. 安裝工具（按順序）
Node.js v24

下載 Node.js 官網 — 選 v24 LTS
或用 winget：winget install OpenJS.NodeJS
pnpm v9


npm install -g pnpm@9
Sui CLI v1.72.1

到 Sui Releases 下載 sui-windows-x86_64.tgz
解壓後把 sui.exe 放到 C:\sui\ 並加入 PATH
或用 Homebrew for Windows (Scoop)：scoop install sui
Git（如果還沒有）


winget install Git.Git
PostgreSQL（backend 用）

下載 PostgreSQL 官網 安裝即可
2. Clone 專案
打開 PowerShell 或 Windows Terminal：


cd C:\projects   # 或你想要的位置
git clone https://github.com/arcoshina/SurveySui.git
cd SurveySui
pnpm install
3. 確認工具正常

node --version    # v24.x.x
pnpm --version    # 9.x.x
sui --version     # 1.72.x
4. 測試 Move 合約

cd contracts
sui move build
sui move test
5. VS Code 設定
安裝 Windows 版 VS Code（不走 Remote-WSL）
打開 C:\projects\SurveySui
推薦安裝的 Extension：Move Analyzer、Sui Move Analyzer
需要 WSL 的情境
只有在需要跑 Linux-only 腳本時才開 WSL，其他開發流程都可以在 Windows 原生完成。

程式碼已全部推到 GitHub，可以安全地在 Windows clone 並繼續開發。
