# SurveyPass 身元トークン

SurveyPass は SurveySui が発行するオンチェーンの通行証で、**譲渡不可**な Soulbound Token です。これにより、アンケート主催者はあなたの個人データを一切露出させることなく「各回答が異なるユーザーから来ている」ことを確認できます。

1冊の SurveyPass には複数の検証を同時に掛けられます（例：まず Email で認証し、後から Google を追加）。証明書の期限が切れたら、再認証すれば延長できます。
SurveyPass はいつでも自分でバーンでき、その際わずかなシビル対策費用が発生する場合があります。

ウォレットの秘密鍵やシードフレーズが漏洩した場合は、ウェブ上で SurveyPass を削除できます。ただし検証ソースのアカウントが漏洩し、そのアカウントを変更できない場合は、当プラットフォームに連絡して SurveyPass の検証を永久に無効化できます。

## 4種類の検証方法

| 方法 | 流れ | 信頼レベル | 有効期間 |
|------|------|----------|--------|
| Email | 認証コード（OTP）を受け取って入力 | Tier 0 | 3か月 |
| Google | OAuth 認可ログイン | Tier 1 | 3か月 |
| GitHub | OAuth 認可ログイン | Tier 1 | 3か月 |
| World ID | World App の実在人物検証 | Tier 2 | 365日 |

Google または GitHub アカウントの利用をおすすめします — 便利で普及しており（一部のアンケートやプラットフォームの代払いポリシーは Tier 1 を要求する場合があります）、同時に Email 検証も得られます。
World Orb は最も強力な実在証明と最長の有効期間を提供します。
Email はシビル対策の要求が高くないアンケートに適しており、無いよりは有る方が良いレベルです。

## プライバシー設計：チェーン上にあなたの個人情報はありません

認証の過程で、あなたの Email やソーシャルアカウントは検証のその瞬間だけ使われ、**オンチェーンには保存されません**。チェーンに記録されるのは nullifier と呼ばれる識別子で、実際のアカウント（ソルト付き）を SHA-256 ハッシュ演算したものです：

- 同じアカウントは常に同じ nullifier を算出します。
- nullifier からあなたのアカウントを逆算することはできません。（既知の）量子コンピュータでも不可能です。
- アンケートに回答する際にもう一度ハッシュされるため、同じ nullifier の異なるアンケートでの行動を追跡することもできません（ただしウォレットのオンチェーンのやり取りは追跡できます）。

これが「本当の身元を覆い隠す」仕組みです。

## 鋳造費用と代払い

Pass の鋳造にはオンチェーンのトランザクションが必要です。プラットフォームは各ウォレットに**生涯2回**の無料代払いを提供します（鋳造 + 更新で共通の枠）。そのため一般ユーザーは初期は無料です。枠を超えた後に証明書を更新する必要がある場合は、自分で gas を払う必要があります（アンケート報酬の売却や SUI の購入などの方法で）。

## SurveyPass の削除

自分の SurveyPass はいつでも削除できます（例：ウォレットを変える場合）。注意点：

- 自費で鋳造した SurveyPass：そのまま削除でき、追加費用はありません。
- プラットフォームが代払いで鋳造した SurveyPass：オンチェーンオブジェクトの削除は保存押金の返還（出資者へ返却）を生むため、代払いの Pass はプラットフォーム経由で削除するか、自分でオンチェーンの脱出費用を払って削除します（実際の金額は鋳造時の代払いコストによります）。
- 削除後、**有効な**証明書の nullifier は解放され、同じアカウントで再認証できます。無効化されたアカウントは削除によってやり直すことはできません。
