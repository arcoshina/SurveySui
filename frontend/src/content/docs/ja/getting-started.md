# SurveySui は Sui ブロックチェーン上のアンケート報酬プラットフォームです

## なぜブロックチェーンでアンケートを？

- **報酬の即時支払い**：アンケートの予算は作成時にオンチェーンの金庫（Vault）にロックされ、支払いルールはスマートコントラクトに書き込まれるため、主催者が後から反故にすることはできません。
- **回答のハードルゼロ**：回答者はトークンを先に買う必要がなく、残高0のウォレットでも回答を完了でき、手数料（gas）は主催者が代払いします。
- **設問も回答も暗号化可能**：あなたの回答は送信前にブラウザ内で暗号化され、アンケート主催者だけが復号して読めます。プラットフォームを含む他者は内容を見られません。

## アンケートに回答したい、何を準備すれば？

Sui ウォレットが1つ必要です。
暗号資産ウォレットを使ったことがなければ、**Slush** が良い入口です。まずソーシャル OAuth アカウント（例：Google アカウント）でウォレットを作って回答し、その後ゆっくり web3 の知識を学べます（ただし先延ばしし過ぎはおすすめしません）。

回答者は **事前にトークンを買ったり入金したりする必要はありません**。ウォレット接続後、プラットフォーム上での初期の活動は、プラットフォームやアンケート主催者が gas を補助できます。

## アンケートに回答して報酬を受け取る方法

1. アンケートページで資格要件を確認する。
2. ウォレットを接続する。
3. 身元認証を完了する（アンケートごとに要件が異なります。Google のバインドを推奨します。これで Gmail も同時にバインドされます）。
4. 回答を入力して送信する。gas は主催者が支払います。注意：*個人情報は入力しないでください！*
5. 報酬トークンが自動的にあなたのウォレットへ送られます。

## アンケートを作りたい

報酬付きアンケートを出したい場合は、その仕組みを完全に理解する必要があります。左側のドキュメント一覧の「アンケートの作成」から始めてください。

## 現在の状況

プラットフォームはまだ開発段階で、機能の検証とデモのために testnet 上で動作しています。testnet トークンには現時点で実際の価値はありません。
正式リリースの時期はお知らせをご確認ください。
