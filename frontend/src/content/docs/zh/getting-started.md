# SurveySui 是在 Sui 區塊鏈上的問卷獎勵平台

## 為什麼用區塊鏈做問卷？

- **獎勵即時發放**：問卷預算在建立時就鎖進鏈上金庫（Vault），發獎規則寫在智能合約裡，發起者無法事後反悔。
- **填答零門檻**：填答者不需要先買幣，錢包 0 餘額也能完成填答，手續費（gas）由發起者代付。
- **問答皆可加密**：你的答案在送出前就在瀏覽器加密，只有問卷發起者能解密閱讀，平台與其他人都看不到內容。

## 我想填問卷，需要準備什麼？

一個 Sui 錢包。  
如果你從來沒用過加密貨幣錢包，**Slush** 是個好開始：你可以先用社群 OAuth 帳號 (例如Google 帳號) 建立錢包並填答，之後再慢慢了解 web3 知識 (但不建議拖太久)。

填答者 **不需要事先買幣或入金** ，連上錢包後的，初期在平台上的活動都可以由平台或問卷方補貼 gas。

## 如何填問卷拿獎勵

 1. 在問卷頁面確認資格要求。
 2. 連接錢包。
 3. 完成身分認證（不同問卷有不同要求，推薦綁 google ，這樣同時也會綁 Gmail）。
 4. 填寫並送出答卷，gas 由問卷方支付。 注意: *不要填寫私人資訊!*
 5. 獎勵代幣將會自動轉到你的錢包。


## 我想發問卷

如果您想發有獎問卷，那麼您將需要完全了解運作機制。請從左側文件列表中的「建立問卷」開始。
