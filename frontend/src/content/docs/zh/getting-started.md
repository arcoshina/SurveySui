# 開始使用

SurveySui 是建立在 Sui 區塊鏈上的問卷獎勵平台：**發起者**用透明、可驗證的方式設定預算與獎勵，**填答者**完成問卷後由智能合約即時發放獎勵——不需等待人工審核，也不擔心獎勵跳票。

## 為什麼用區塊鏈做問卷？

- **獎勵保證發放**：問卷預算在建立時就鎖進鏈上金庫（Vault），發獎規則寫在智能合約裡，發起者無法事後反悔。
- **填答零門檻**：填答者**不需要持有任何加密貨幣**，錢包餘額為 0 也能完成認證、填答、領獎，手續費（gas）由問卷方或平台代付。
- **一人一票**：透過鏈上身分證明（SurveyPass）防止同一人重複填答或大量假帳號灌水，確保資料品質。
- **答案加密**：你的答案在送出前就在瀏覽器加密，只有問卷發起者能解密閱讀，平台與其他人都看不到內容。

## 三種角色

| 角色 | 做什麼 | 需要什麼 |
|------|--------|----------|
| 發起者 | 建立問卷、注入獎勵預算、查看結果 | Sui 錢包與少量 SUI |
| 填答者 | 完成身分認證、填答、領取獎勵 | Sui 錢包（餘額可為 0） |
| 平台 | 代付手續費、簽發認證憑證、到期清理 | — |

## 我需要準備什麼？

唯一需要的是一個 Sui 錢包。如果你從來沒用過加密貨幣錢包，推薦 **Slush**（Sui 官方團隊出品）：用 Google 帳號即可建立，免記助記詞、免安裝門檻，幾十秒完成。當然，任何支援 Sui 的錢包（如瀏覽器擴充功能版本）都可以使用。

填答者**不需要**事先買幣或入金——連上錢包之後的鑄造身分證明、送出問卷，都可以由平台或問卷方代付 gas。

## 快速上手

**我想發問卷**：

1. 連接錢包，進入「建立問卷」。
2. 用 Markdown 編輯器設計題目（單選、多選、文字、量表）。
3. 設定獎勵金額、名額與截止時間。
4. 確認費用後簽署一筆交易，問卷與預算同時上鏈。
5. 分享問卷連結，在儀表板追蹤填答進度。

詳細步驟請看「建立問卷」篇。

**我想填問卷拿獎勵**：

1. 打開問卷連結，連接錢包。
2. 第一次使用需完成一次身分認證（Email、Google、GitHub 或 World ID 擇一）。
3. 填答並送出——只需要簽一次名，gas 已由問卷方或平台代付。
4. 獎勵代幣即時發到你的錢包。

詳細步驟請看「填答與領獎」篇；認證方式的差異請看「身分認證與 SurveyPass」篇。

## 目前狀態

平台目前運行於 Sui **開發測試網（devnet）**，用於功能驗證與展示。測試網的代幣沒有實際價值，且鏈上資料可能隨網路重置而清空；正式上線時間請關注公告。
