# 建立問卷

完整流程：設計問卷 -> 設定獎勵 -> 配資上鏈 -> 管理進度 -> 結案收尾。

## 第一步：設計問卷

問卷使用 Markdown 編輯器撰寫，支援四種題型：

- 單選題：受訪者擇一作答。
- 多選題：可複選。
- 文字題：自由輸入。
- 量表題：例如 1–5 分滿意度。

選擇題支援最多 50 個選項。  
系統也支援隨機排列選項，降低排序偏誤。

您可以選擇將問卷題目在 *本地端加密* 後才上鏈。 10 KB 以內的問卷題目將直接存在鏈上，較大的問卷將自動改存 Walrus，因此您需要為大型問卷準備 Walrus 代幣作為儲存費用。無論哪種方式，問卷的雜湊值都會與金庫綁定。**問卷一旦發布就不可修改**。

## 第二步：設定參與條件與獎勵

截止時間：問卷有效期最長 3 個月，到期自動停止收件。
受眾驗證來源：你可以選擇接受持有哪一些驗證來源（Email、Google、GitHub、World ID）的使用者，或要求持有特定 NFT。要求越強的認證（如 World Orb），越難用假帳號灌水，但是受眾也越少。

獎勵規則：
- 單次填答獎勵金額（每位首次填答者領取的數額）。
- 最多接受名額（預算上限 = 名額 × 單次獎勵）。
- 重複填答（選用）：允許同一人再次填答並領取較低的重複獎勵，適合追蹤型調查。

## 第三步：配資上鏈

設定完成後進入配資頁，系統會列出預估總成本：
確認後，以下動作會在同一次交互完成（全部成功或整筆取消，不會出現「某幾步失敗」的中間狀態）：

1. 投入 SUI 兌換獎勵代幣 SSR。
2. 建立問卷專屬金庫（Vault）並注入SSR獎金。
3. 注入 gas 補償儲備。
4. 繳納協議費 20% ，目前半價優惠。
5. 註冊問卷內容（雜湊與金庫綁定）。
6. 開放金庫供大眾填答。

完成後你會獲得問卷連結，直接分享即可。

提示: 如果問卷的題目也有加密，網址中 # 後面的部分就是解密金鑰，請注意不要隨意外流


## 第四步：結案與收尾

儀表板即時顯示：已填人數、完成率、已發放獎勵、剩餘額度、截止倒數。

結案（Close）：發起者可隨時結案；結案後剩餘的獎勵預算與 gas 儲備會退回你的錢包，問卷停止收件。截止時間過後，任何人都可以觸發結案，但資金一樣退還給你。

查看結果：在結果頁用你的金鑰解密所有答案，內建基本統計。

銷毀：結案後可銷毀問卷的題目和**答案**。銷毀以每批 500 筆答案分批執行，最後一批銷毀金庫與問卷物件，並回收占用鏈上儲存的押金（storage rebate）。若您一直放著不管，平台會在寬限期過後自動清理。扣除執行成本後的盈餘將*收取 50% 作為手續費*，請務必記得銷毀資料。
