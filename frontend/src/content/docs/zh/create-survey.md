# 建立問卷

這一篇帶你走完發起者的完整流程：設計問卷 → 設定獎勵 → 配資上鏈 → 管理進度 → 結案收尾。

## 第一步：設計問卷

問卷使用 Markdown 編輯器撰寫，支援四種題型：

- **單選題**：受訪者擇一作答。
- **多選題**：可複選。
- **文字題**：自由輸入。
- **量表題**：例如 1–5 分滿意度。

每題選項最多 50 個。填答時系統會自動隨機排列選項順序，降低排序偏誤。

問卷內容會在你的瀏覽器**加密後**才上鏈：內容較小（10 KB 以內）直接存在鏈上，較大的問卷自動改存去中心化儲存網路 Walrus，鏈上只記錄索引。無論哪種方式，問卷內容的雜湊值（指紋）都會與金庫綁定——**問卷一旦發布就不可竄改**，受訪者填到的內容保證與你發布的一致。

## 第二步：設定參與條件與獎勵

**截止時間**：問卷有效期最長 3 個月（鏈上強制上限 92 天），到期自動停止收件。

**受眾驗證來源**：你可以指定填答者必須通過哪些身分認證（Email、Google、GitHub、World ID），或要求持有特定 NFT。要求越強的認證（如 World ID 真人驗證），假帳號灌水的成本越高。

**獎勵規則**：

- 單次填答獎勵金額（每位首次填答者領取的數額）。
- 最多接受名額（預算上限 = 名額 × 單次獎勵）。
- 重複填答（選用）：允許同一人再次填答並領取較低的重複獎勵，適合追蹤型調查。

## 第三步：配資上鏈

設定完成後進入配資頁，系統會列出**預估總成本**：

| 項目 | 說明 |
|------|------|
| 獎勵預算 | 名額 × 單次獎勵（含重複填答額度） |
| 協議費 | 預算的 10% |
| Gas 補償預留 | 為填答者代付手續費的儲備金 |
| Walrus 儲存費 | 僅問卷超過 10 KB 時產生 |

確認後**只需簽署一筆交易**，以下動作會原子完成（要嘛全部成功、要嘛全部取消，不會出現「錢扣了問卷沒建好」的中間狀態）：

1. 投入 SUI 兌換獎勵代幣 SSR。
2. 建立問卷專屬金庫（Vault）並注入預算。
3. 注入 gas 補償儲備。
4. 繳納協議費。
5. 註冊問卷內容（雜湊與金庫綁定）。
6. 開放金庫供大眾填答。

完成後你會獲得問卷連結，直接分享即可。

## 第四步：進行中管理

儀表板即時顯示：已填人數、完成率、已發放獎勵、剩餘額度、截止倒數。

進行中你仍可調整部分參數（僅發起者可操作）：gas 補償金額、到期後的自動銷毀寬限期等。若想提前停止收件但暫不動資金，可將問卷**封存（archive）**。

## 第五步：結案與收尾

**結案（Close）**：發起者可隨時結案；結案後剩餘的獎勵預算與 gas 儲備**全額退回**你的錢包，問卷停止收件。若你一直不動作，任何人都可以在截止時間過後代為觸發結案（資金一樣只會退給你）。

**查看結果**：在結果頁用你的金鑰解密所有答案，內建基本統計（選項分布、量表平均等）。

**銷毀（Purge）**：結案後可銷毀問卷，分批刪除鏈上答案資料並回收物件。Sui 會將當初占用儲存空間的押金退還（storage rebate），扣除執行成本後的盈餘你可分得 50%——**清理過期問卷不只是好習慣，還能拿回一部分錢**。你結案後即可銷毀；若放著不管，平台會在寬限期（預設 92 天）過後自動清理。

## 給進階使用者

- 協議費率：`effective_fee_bps = 1000`（10%），計費基準為注入預算。
- 問卷內容分流門檻：加密後 10 KB（`VITE_SURVEY_SIZE_THRESHOLD_KB`）；超過走 Walrus，鏈上存 `blob_id`。
- 答案 inline 上限預設 6 KB，發起者可在 1–32 KB 間調整；超過走 Walrus。
- 金庫與問卷物件在鏈上 1:1 綁定，同一份內容雜湊全域只能註冊一次。
- 銷毀為分批執行（每批預設 100 筆答案），最後一批銷毀金庫與問卷物件並觸發 storage rebate。
- 銷毀權限分三層：**發起者**結案後可立即銷毀；**平台代付帳戶（sponsor）與合約 admin** 可在截止時間加寬限期（最短 7 天、預設 92 天）後觸發；**其他任何人**則須等到期後 365 天，作為平台失能時的 liveness 備援。
- 獎勵代幣 SSR 採儲備池定價：投入的 SUI 進入池中，依池內儲備比率鑄造 SSR（空池初始匯率 1 SUI = 1000 SSR）。
