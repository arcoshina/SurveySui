# SurveyPass 誰位通證

SurveyPass 是 SurveySui 製作的鏈上通行證：一個**不可轉讓**的 Soulbound Tokens。它讓問卷發起者能確認「每份答卷來自不同使用者」，同時不暴露您的任何個人資料。

一本 SurveyPass 可以同時掛多個驗證（例如先用 Email 認證、之後再補 Google），憑證到期後重新認證即可延長。
SurveyPass 在任何時候皆可自行銷毀，可能會產生微小的防女巫費用。  

當錢包的私鑰或註記詞洩漏時，可以到網頁刪除 SurveyPass。但若驗證來源帳號洩漏、而您又無法更換帳號時，可以聯絡本平台永久註銷 SurveyPass 的驗證。

## 四種驗證方式

| 方式 | 流程 | 信任等級 | 有效期 |
|------|------|----------|--------|
| Email | 收驗證碼（OTP）回填 | Tier 0 | 3 個月 |
| Google | OAuth 授權登入 | Tier 1 | 3 個月 |
| GitHub | OAuth 授權登入 | Tier 1 | 3 個月 |
| World ID | World App 真人驗證 | Tier 2 | 365 天 |

推薦使用 Google 或 GitHub 帳號 — 方便又普遍（部分問卷或平台代付政策可能要求 Tier 1），也同時取得 Email 驗證。  
World Orb 提供最強的真人證明與最長效期。  
Email 適合對防女巫要求不高的問卷，屬於有比沒有好的等級。  


## 隱私設計：鏈上沒有您的個資

認證流程中，您的 Email 或社群帳號只在驗證的當下使用，**不會被上鏈儲存**。鏈上記錄的是一個叫 nullifier 的識別碼——由真實帳號（加鹽）經 SHA-256 雜湊運算而來：

- 同一個帳號永遠算出同一個 nullifier。
- 從 nullifier 無法反推出您的帳號，就算是（已知的）量子電腦也不行。
- 填問卷時還會再雜湊一次，因此也不能追蹤同一個 nullifier 在不同問卷的行為 (但可以追蹤錢包的鏈上交互)。

這就是「遮蓋真實身分」的機制。


## 鑄造費用與代付

鑄造 Pass 需要一筆鏈上交易。平台為每個錢包提供**終身 2 次**的免費代付（鑄造 + 更新共用額度），所以一般使用者初期免費。超過額度後若需更新憑證，則需要自己付 gas (透過出售問卷獎勵或購買 SUI 之類的方法)。

## 刪除 SurveyPass

您隨時可以刪除自己的 SurveyPass（例如要換錢包）。要注意：

- 自付鑄造的 SurveyPass：直接刪除，無額外費用。
- 平台代付鑄造的 SurveyPass：因為刪除鏈上物件會產生儲存押金返還（會退到出資方），代付的 Pass 需透過平台代刪，或自行支付鏈上逃生費刪除（實際金額依鑄造時的代付成本而定）。  
- 刪除後，**有效**憑證的 nullifier 會被釋放，同一帳號可以重新認證；被註銷的帳號無法靠刪除重來。
