# 설문 만들기

전체 흐름: 설문 설계 -> 보상 설정 -> 자금 주입 후 온체인 공개 -> 진행 관리 -> 종료 및 마무리.

## 1단계: 설문 설계

설문은 Markdown 에디터로 작성하며, 네 가지 문항 유형을 지원합니다:

- 단일 선택: 답변자가 하나를 고릅니다.
- 다중 선택: 복수 선택이 가능합니다.
- 텍스트: 자유 입력.
- 척도: 예를 들어 1~5점 만족도.

선택형 문항은 최대 50개 선택지를 지원합니다.
선택지의 무작위 배열도 지원하여 순서로 인한 편향을 줄입니다.

설문 문항을 *로컬에서 암호화* 한 뒤 온체인에 올리도록 선택할 수 있습니다. 10 KB 이내의 문항은 그대로 온체인에 저장되고, 더 큰 설문은 자동으로 Walrus 에 저장되므로, 큰 설문을 위해 저장 비용으로 Walrus 토큰을 준비해야 합니다. 어느 방식이든 설문의 해시 값은 금고에 바인딩됩니다. **설문은 일단 게시되면 수정할 수 없습니다.**

## 2단계: 참여 조건과 보상 설정

마감 시간: 설문의 유효 기간은 최장 3개월이며, 만료되면 자동으로 접수를 중단합니다.
허용할 검증 소스: 어떤 검증 소스(Email, Google, GitHub, World ID)를 가진 사용자를 받을지 선택하거나, 특정 NFT 보유를 요구할 수 있습니다. 요구하는 인증이 강할수록(예: World Orb) 가짜 계정으로 부정 응답하기 어려워지지만, 대상자도 줄어듭니다.

보상 규칙:
- 1회 답변당 보상 금액(최초 답변자가 받는 액수).
- 최대 수용 인원(예산 상한 = 인원 × 1회 보상).
- 반복 답변(선택): 같은 사람이 다시 답변하고 더 낮은 반복 보상을 받도록 허용하며, 추적형 조사에 적합합니다.

## 3단계: 자금 주입 후 온체인 공개

설정이 완료되면 자금 주입 페이지로 들어가고, 시스템이 예상 총비용을 표시합니다.
확인하면 다음 동작이 동일한 인터랙션 안에서 완료됩니다(모두 성공하거나 통째로 취소되며, '일부 단계만 실패'하는 중간 상태는 발생하지 않습니다):

1. SUI 를 투입해 보상 토큰 SSR 을 발행합니다.
2. 설문 전용 금고(Vault)를 생성하고 SSR 보상을 주입합니다.
3. gas 보상 리저브를 주입합니다.
4. 프로토콜 수수료 20% 를 납부합니다(현재 반값).
5. 설문 콘텐츠를 등록합니다(해시를 금고에 바인딩).
6. 금고를 대중의 답변에 개방합니다.

완료되면 설문 링크를 받게 되며, 그대로 공유할 수 있습니다.

팁: 설문 문항도 암호화한 경우, URL 의 `#` 뒤 부분이 복호화 키입니다. 함부로 외부에 유출하지 않도록 주의하세요.

## 4단계: 종료 및 마무리

대시보드는 실시간으로 표시합니다: 답변 인원, 완료율, 지급된 보상, 잔여 예산, 마감 카운트다운.

종료(Close): 발행자는 언제든 종료할 수 있습니다. 종료 후 남은 보상 예산과 gas 리저브는 당신의 지갑으로 반환되고, 설문은 접수를 중단합니다. 마감 시간이 지나면 누구나 종료를 트리거할 수 있지만, 자금은 마찬가지로 당신에게 반환됩니다.

결과 보기: 결과 페이지에서 당신의 키로 모든 답변을 복호화하며, 기본 통계가 내장되어 있습니다.

소각: 종료 후 설문의 문항과 **답변**을 소각할 수 있습니다. 소각은 답변 500건 단위 배치로 분할 실행되며, 마지막 배치에서 금고와 설문 객체를 소각하고 온체인 저장에 점유한 보증금(storage rebate)을 회수합니다. 계속 방치하면 플랫폼이 유예 기간 경과 후 자동으로 정리합니다. 실행 비용을 차감한 잉여는 *50% 를 수수료로 징수* 하므로, 반드시 데이터 소각을 잊지 마세요.
