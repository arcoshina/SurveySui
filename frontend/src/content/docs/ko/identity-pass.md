# SurveyPass 신원 토큰

SurveyPass 는 SurveySui 가 만든 온체인 통행증으로, **양도 불가능한** Soulbound Token 입니다. 이는 설문 발행자가 당신의 어떤 개인 데이터도 노출하지 않으면서 '각 답변이 서로 다른 사용자에게서 왔음'을 확인할 수 있게 합니다.

한 권의 SurveyPass 에는 여러 검증을 동시에 걸 수 있습니다(예: 먼저 Email 로 인증하고, 이후 Google 을 추가). 증명서가 만료되면 다시 인증해 연장할 수 있습니다.
SurveyPass 는 언제든 스스로 소각할 수 있으며, 이때 아주 작은 시빌 방지 비용이 발생할 수 있습니다.

지갑의 개인 키나 시드 구문이 유출된 경우, 웹에서 SurveyPass 를 삭제할 수 있습니다. 다만 검증 소스 계정이 유출되었고 그 계정을 바꿀 수 없다면, 본 플랫폼에 연락하여 SurveyPass 의 검증을 영구히 무효화할 수 있습니다.

## 네 가지 검증 방법

| 방법 | 흐름 | 신뢰 등급 | 유효 기간 |
|------|------|----------|--------|
| Email | 인증 코드(OTP)를 받아 입력 | Tier 0 | 3개월 |
| Google | OAuth 인가 로그인 | Tier 1 | 3개월 |
| GitHub | OAuth 인가 로그인 | Tier 1 | 3개월 |
| World ID | World App 실제 인물 검증 | Tier 2 | 365일 |

Google 또는 GitHub 계정 사용을 권장합니다 — 편리하고 보편적이며(일부 설문이나 플랫폼 대납 정책은 Tier 1 을 요구할 수 있습니다), 동시에 Email 검증도 얻습니다.
World Orb 는 가장 강력한 실제 인물 증명과 가장 긴 유효 기간을 제공합니다.
Email 은 시빌 방지 요구가 높지 않은 설문에 적합하며, 없는 것보다는 나은 등급입니다.

## 프라이버시 설계: 체인 위에 당신의 개인정보는 없습니다

인증 과정에서 당신의 Email 이나 소셜 계정은 검증하는 그 순간에만 사용되며, **온체인에 저장되지 않습니다**. 체인에 기록되는 것은 nullifier 라는 식별자로, 실제 계정(솔트 추가)을 SHA-256 해시 연산한 것입니다:

- 같은 계정은 항상 같은 nullifier 를 산출합니다.
- nullifier 로부터 당신의 계정을 역산할 수 없으며, (알려진) 양자 컴퓨터로도 불가능합니다.
- 설문에 답할 때 한 번 더 해시되므로, 같은 nullifier 의 서로 다른 설문에서의 행동도 추적할 수 없습니다(다만 지갑의 온체인 인터랙션은 추적할 수 있습니다).

이것이 '실제 신원을 가리는' 메커니즘입니다.

## 발행 비용과 대납

Pass 발행에는 온체인 트랜잭션이 필요합니다. 플랫폼은 각 지갑에 **평생 2회**의 무료 대납을 제공합니다(발행 + 갱신이 공용 한도). 그래서 일반 사용자는 초기에 무료입니다. 한도를 초과한 후 증명서를 갱신해야 하면, 직접 gas 를 내야 합니다(설문 보상 판매나 SUI 구매 등의 방법으로).

## SurveyPass 삭제

자신의 SurveyPass 는 언제든 삭제할 수 있습니다(예: 지갑을 바꿀 때). 주의할 점:

- 자비로 발행한 SurveyPass: 그대로 삭제하며 추가 비용이 없습니다.
- 플랫폼 대납으로 발행한 SurveyPass: 온체인 객체를 삭제하면 저장 보증금 환급(출자자에게 반환)이 발생하므로, 대납된 Pass 는 플랫폼을 통해 삭제하거나, 직접 온체인 탈출 비용을 지불해 삭제합니다(실제 금액은 발행 시점의 대납 비용에 따릅니다).
- 삭제 후, **유효한** 증명서의 nullifier 가 해제되어 같은 계정으로 다시 인증할 수 있습니다. 무효화된 계정은 삭제로 다시 시작할 수 없습니다.
