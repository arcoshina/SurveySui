# SurveySui 는 Sui 블록체인 위의 설문 보상 플랫폼입니다

## 왜 블록체인으로 설문을 하나요?

- **보상 즉시 지급**: 설문 예산은 생성 시점에 온체인 금고(Vault)에 잠기고, 지급 규칙은 스마트 컨트랙트에 기록되므로 발행자가 나중에 번복할 수 없습니다.
- **답변 진입 장벽 제로**: 답변자는 토큰을 먼저 살 필요가 없으며, 잔액 0 인 지갑으로도 답변을 완료할 수 있고, 수수료(gas)는 발행자가 대납합니다.
- **질문과 답변 모두 암호화 가능**: 당신의 답변은 전송 전에 브라우저에서 암호화되어, 설문 발행자만 복호화해 읽을 수 있고, 플랫폼을 포함한 다른 누구도 내용을 볼 수 없습니다.

## 설문에 답하고 싶은데 무엇을 준비해야 하나요?

Sui 지갑 하나가 필요합니다.
암호화폐 지갑을 써본 적이 없다면 **Slush** 가 좋은 출발점입니다. 먼저 소셜 OAuth 계정(예: Google 계정)으로 지갑을 만들어 답변하고, 이후 천천히 web3 지식을 익혀가면 됩니다(다만 너무 미루는 것은 권장하지 않습니다).

답변자는 **미리 토큰을 사거나 입금할 필요가 없습니다**. 지갑 연결 후, 플랫폼에서의 초기 활동은 플랫폼이나 설문 발행자가 gas 를 보조할 수 있습니다.

## 설문에 답하고 보상을 받는 방법

1. 설문 페이지에서 자격 요건을 확인합니다.
2. 지갑을 연결합니다.
3. 신원 인증을 완료합니다(설문마다 요건이 다르며, Google 바인딩을 권장합니다. 이렇게 하면 Gmail 도 함께 바인딩됩니다).
4. 답변을 작성해 제출합니다. gas 는 발행자가 지불합니다. 주의: *개인정보를 입력하지 마세요!*
5. 보상 토큰이 자동으로 당신의 지갑으로 전송됩니다.

## 설문을 만들고 싶어요

보상형 설문을 내고 싶다면 그 작동 방식을 완전히 이해해야 합니다. 왼쪽 문서 목록의 '설문 만들기'부터 시작하세요.

## 현재 상태

플랫폼은 아직 개발 단계로, 기능 검증과 시연을 위해 testnet 에서 운영됩니다. testnet 토큰은 현재 실제 가치가 없습니다.
정식 출시 시기는 공지를 확인해 주세요.
