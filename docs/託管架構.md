# 託管架構（內部備忘）

> 詳見 [安全指引.md](./安全指引.md)、[V4_Tasks.md](./V4_Tasks.md)

## 運行（單一平台：Cloudflare + GitHub CI）

| 元件 | 託管 |
|------|------|
| 前端 | CF Pages |
| Gas 代付 | CF Worker `gas-station` + DO |
| API（原 BFF） | 遷移中 → CF Worker `api`；過渡期 SaaS |
| DB | 單一 CF D1（revocation + sponsor ledger） |
| 定時 purge | CF Cron |
| 程式/CI | GitHub Actions → wrangler deploy |

## 秘密

- `wrangler secret` + BFF SaaS secrets（過渡）
- **Tier0** `SUI_ADMIN_PRIVATE_KEY`：僅部署腳本，永不進 BFF/Worker

## 金鑰（Phase 2-lite）

| 用途 | 變數 | 持有者 |
|------|------|--------|
| Ticket 簽名 | `SURVEY_PASS_ISSUER_PRIV` | BFF only |
| 代付 multisig K1/K2 | `GAS_SPONSOR_PRIV_1/2` | Gas Worker + BFF 代付路徑 |
| 代付 K3 | 離線備份 | 不進 runtime |

## 環境

- testnet 公開測 → 穩定後 mainnet
- 兩套 env / secrets 完全隔離；mainnet 前重設（安全指引 §8）
